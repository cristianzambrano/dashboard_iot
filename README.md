# Dashboard IoT UTEQ 2026

Este proyecto es una interfaz web estática para el monitoreo visual de dispositivos IoT. Presenta un panel con indicadores de temperatura, humedad, presión y estado del dispositivo, además de un historial de lecturas en formato de tarjetas y tabla.

## Características

- Diseño responsive para pantallas de escritorio y móviles.
- Sección de tarjetas con valores actuales del sensor.
- Bloque de métricas históricas destacadas.
- Tabla de lecturas IoT con fecha, hora, dispositivo, temperatura, humedad y estado.
- Estilos personalizados en CSS para una apariencia moderna.

## Estructura del proyecto

- `index.html`: estructura principal de la interfaz.
- `css/styles.css`: estilos del dashboard.
- `img/`: carpeta para imágenes del proyecto.
- `js/`: carpeta para scripts JavaScript futuros.

## Requisitos

No se requiere instalación de dependencias para visualizar el proyecto. Solo necesitas un navegador web moderno.

## Cómo ejecutar

1. Abre el archivo `index.html` en tu navegador.
2. O bien, si prefieres, puedes usar una extensión como Live Server para ver los cambios en tiempo real.

## Tecnologías utilizadas

- HTML5
- CSS3
